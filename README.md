    复制ctrl+c
    粘贴ctrl+v
    查找ctrl+f
    替换ctrl+h
    撤销ctrl+z
    重做ctrl+y

    自动保存（每隔一段时间进行自动保存 可选择是否开启此项功能）

    设置字体大小
